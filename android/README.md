# Android (natif)

Ce dossier contient le squelette du projet Android natif.

## Démarrage rapide

1. Ouvrez le dossier `android/` dans Android Studio.
2. Android Studio téléchargera automatiquement le wrapper Gradle manquant si besoin.
3. Lancez l'application sur un émulateur ou un appareil.

## Points d'entrée

- `app/src/main/java/com/tabata/app/MainActivity.kt`
- `app/src/main/res/layout/activity_main.xml`
